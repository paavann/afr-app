# Checkpoints Directory

Place your trained UV-Net `best.ckpt` file here.

The checkpoint is generated by training the UV-Net segmentation model:

```bash
cd /path/to/UV-Net
conda run -n uvnet_env python segmentation.py train \
    --dataset mfcad \
    --dataset_path /path/to/mfcad_data \
    --max_epochs 100 \
    --batch_size 64 \
    --experiment_name feature_segmentation
```

The best checkpoint will be saved at:
`results/feature_segmentation/<date>/<time>/best.ckpt`

Copy it here:
```bash
cp results/feature_segmentation/*/best.ckpt /path/to/afr-app/api/checkpoints/
```

> **Note:** Without a checkpoint, the API runs in **stub mode** — it returns
> mock predictions so you can test the full pipeline end-to-end.
